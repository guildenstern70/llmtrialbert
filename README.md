## Setup Conda

Add PyTorch:

    conda install -c pytorch pytorch torchvision 

Add Transformers:

    conda install -c conda-forge transformers

Add SciPy

    conda install -c anaconda scipy

Add PlotLy

    conda install -c plotly plotly
